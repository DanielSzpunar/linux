#1: How do we long onto another computer remotely using SSH i terminal?
ssh <username>@<remote>

	to include the port: ssh <username>@<remote> -p 2220

	For example: myUserName@theWebsitesName.org -p 2220

#2: cat
			This allows us to see the contents of a file.

			For example: cat textFileName.txt
				This will print the contents of the file to the screen.

#3 cat ./-
		This allows us to access a file that does not follow a conventional naming method.

#4 cat this\ file\ has\ spaces\
		This allows us to access a file that has spaces.

#5 ls -a
	ls shows the files and directories in list format.
	ls -a shows all files including hidden files.

#6 file ./*
	This shows the type of file in the current directory.
	if the file is data then it is unreadable. if it is ASCII text then it is readable.
	This is useful to find readable files in a directory to pull information.

#7 find -size 1033c
	This searches for files with the size of 1033 bytes.
